About

I have experience developing applications in Python, particularly focusing on desktop applications that combine graphical interfaces with data management. In my projects, I usually organize the code in a structured way, separating the parts responsible for the user interface, the program logic, and the data storage. This approach helps keep the project organized and makes the system easier to maintain and expand.

For building graphical interfaces, I have worked with both Tkinter and Kivy. Using Tkinter allowed me to understand the fundamentals of creating desktop interfaces in Python, such as managing windows, buttons, tables, and handling user interactions. I have used it to develop functional interfaces for systems where users can interact with data directly through forms and visual components.

I have also worked with Kivy, which provides a more flexible way to design interfaces and keep the visual layout separated from the program’s internal logic. This allows the structure of the interface to be defined in dedicated layout files, making the design cleaner and easier to modify. The layout structure follows an approach similar to how styles are organized in web development, which helps keep the visual components well structured.

In terms of data management, I have experience working with CSV and JSON files to store and process information. CSV files are useful for managing structured records such as product lists, inventory data, or sales logs, while JSON allows me to handle structured data in a flexible format that can be easily read and written by applications.

Additionally, I have experience using SQL to manage relational databases. This includes creating tables, inserting records, retrieving specific data through queries, and organizing information in a structured way. Working with SQL has helped me understand how to manage larger datasets and structure information more efficiently.

Overall, my experience combines Python programming, graphical interface development, and data management, allowing me to build complete applications that handle both user interaction and structured data processing. I focus on keeping projects organized and modular so that different parts of the system can work together clearly and efficiently.
