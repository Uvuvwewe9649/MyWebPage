Point of Sale (POS) System

Python / Kivy

About

Designed and developed a desktop Point of Sale system focused on inventory control and sales management. The application was built using Python with the Kivy framework to create a responsive graphical interface and modular architecture.

Key Features

  • Product search with real-time cart management
  • Dynamic quantity adjustment through intuitive controls
  • Inventory management and tracking modules
  • Employee registration and authentication system
  • Administrative operations dashboard
  • Persistent data handling for products and inventory

Architecture

Structured using a modular folder architecture that separates UI (.kv) files from logic (.py) files, improving maintainability and scalability. The workflow is optimized to reduce user input and improve transaction speed.
